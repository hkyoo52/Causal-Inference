## 복습
* 처치효과 : $\kappa = Y_{1i}-Y_{0i}$
* 처치효과는 알 수 X
* $Y_i = Y_{0i} + T_i(Y_{1i}-Y_{0i})$
* $ATE = E[Y_1-Y_0]$

 ## 선형 회귀
 #### 예시
 * 온라인 교육과 시험성적의 상관관계
  * 대면 수업 : 0, 온라인 수업 : 1
 * $exam_i = \beta_0 + k Online_i + u_i$
   * $\Rightarrow Y_i = \beta_0 + k T_i + u_i$
* $E[Y|T=0] = \delta_0$
* $E[Y|T=1] = \delta_0 + k$
* k = ATE
